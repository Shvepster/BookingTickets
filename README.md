# 🎟️ Ticket Booking System (Система бронирования билетов)

## 🗄 Предметная область и Модель данных

В системе реализовано 5 ключевых сущностей:

1. `User` (Пользователь)
2. `Event` (Мероприятие - концерт, спектакль и т.д.)
3. `Venue` (Площадка - театр, стадион)
4. `Ticket` (Билет)
5. `Category` (Категория/Тег - "Рок", "Комедия", "18+")
